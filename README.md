# Funcionários
## Descrição do projeto
Um projeto Flutter de funcionários aleatórios com navegação.

## Tecnologias

Aplicativos utilizados:
- VSCode
- Android Studio
- Flutter

## Passo a passo de como executar
1. Clone o repositório
2. Abra com VsCode, Abra o terminal CTRL + ", execute o comando **flutter pub get** para instalar as dependências
3. Navegue até o arquivo **lib/main.dart** e dê play ou execute o comando flutter run para rodar o projeto
4. Escolha navegador ou um emulador para testar, ou abra o arquivo /lib/main.dart e clique em Play.
5. O projeto irá abrir a tela de Splash com uma animação, navegue pelos funcionários.
